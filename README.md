# FloatFolder - 悬浮桌面文件夹工具

一个轻量级的桌面悬浮窗工具，实时展示指定文件夹中文件的变化，支持拖拽文件到任意位置、点击复制、图片预览等功能。

## 功能

- [x] 悬浮窗展示文件夹内容，支持窗口置顶
- [x] 多文件夹 Tab 标签页切换
- [x] 实时监控文件变化（新增/删除/修改自动刷新）
- [x] 拖拽文件到桌面或任意输入框
- [x] 单击文件复制到剪贴板
- [x] 双击文件用系统默认程序打开
- [x] 右键菜单（打开/复制/复制路径/在资源管理器中显示）
- [x] 图片文件内联缩略图展示（无需悬停即可查看）
- [x] 亮色/暗色主题切换（Apple 设计规范，红绿灯按钮、Spotlight 搜索、Finder 文件列表）
- [x] 图片文件悬停大图预览
- [x] 文件搜索过滤
- [x] 窗口尺寸和位置自动记忆
- [x] 系统托盘，关闭窗口隐藏到托盘
- [x] 开机自启动（可设置）
- [x] 根据文件类型显示彩色图标
- [x] 悬浮图标模式（Apple 风格呼吸灯图标，降低视觉干扰）
- [x] 鼠标悬停展开完整界面，移开自动收起
- [x] 设置面板（右键图标 / 标题栏 / 托盘菜单打开），支持：置顶、开机自启、透明度、主题、全局快捷键

## 技术栈

| 层 | 技术 |
|----|------|
| 框架 | Electron |
| 前端 | React + TypeScript |
| 样式 | TailwindCSS |
| 文件监听 | chokidar |
| 持久化 | electron-store |
| 构建 | Vite + electron-builder |

## 环境变量

无需配置环境变量。

## 安装与运行

```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 构建可执行文件（Windows）
npm run electron:build
```

## 使用说明

1. 启动后点击「添加文件夹」选择要监控的文件夹
2. 文件夹内容会实时展示在悬浮窗中
3. **单击** 文件 → 复制到剪贴板
4. **双击** 文件 → 用默认程序打开
5. **拖拽** 文件 → 拖到桌面或其他程序
6. **右键** 文件 → 更多操作菜单
7. **悬停** 图片文件 → 显示大图预览
8. **悬停** 悬浮图标 → 展开完整界面，移开自动收起
9. **右键** 悬浮图标 / 点击标题栏「设置」/ 托盘菜单「设置」→ 打开设置面板
10. 窗口关闭后最小化到系统托盘，双击托盘图标恢复
